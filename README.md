# Savings Goal Tracker

A simple and intuitive web application to help you track your savings goals and monitor your progress toward financial objectives.
**Disclaimer**: This is still a work in progress, but I'm excited to see how it will grow as I continue on my coding journey.

## 🎯 Overview

Savings Goal Tracker is a lightweight web application built with HTML and CSS that allows you to set, visualise, and track your savings goals.  Whether you're saving for a vacation, a new car, or any other financial objective, this tool helps you stay motivated and on track.

## ✨ Features

- **Goal Creation** - Easily set up new savings goals with target amounts
- **Progress Tracking** - Visualise your progress toward each goal
- **Clean Interface** - Minimal, user-friendly design for easy navigation
- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Goal Management** - View, update, or remove your savings goals

## 🛠️ Technologies Used

- **HTML** (63%) - Semantic markup and structure
- **CSS** (37%) - Styling and responsive design
